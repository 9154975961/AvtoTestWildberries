# AvtoTest
### Описание:
---
```sh
Парсер для сайта Wildberries. 
С сайта парсятся:
- Гарнитуры и наушники
- Смартфоны
- Планшеты
- Ноутбуки
- Бытовая техника
    Парсер сначала загоняет в память по 600 наименований товаров в память после чего начинает опрос сайта на наличие обновлений.
    При обнаружении нового товара добавляет уже к имеющейся базе.После проверят стоймость товара ужк с имеющейся базой и если стоймость данного товара меньше 30% от стоймости имеющейся в базе, то отправляет сообщение в Телеграм канал (определенному пользователю), с сылкой на товар.
```
